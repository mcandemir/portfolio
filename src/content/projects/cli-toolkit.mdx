## Overview

A suite of command-line tools designed to streamline common developer workflows. Each tool follows Unix philosophy — do one thing well.

## Tools Included

- **scaffold** — Project template generator with interactive prompts
- **gitclean** — Intelligent branch cleanup with safety checks
- **fswatch** — File watcher with configurable action triggers

## Technical Highlights

Built with Rust for zero-cost abstractions and memory safety. Uses Tokio for async I/O in the file watcher component, and Clap for ergonomic CLI argument parsing.
