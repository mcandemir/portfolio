## Overview

A personal portfolio and blog built for maximum performance and developer experience. Uses Astro's static output mode with React islands for interactive components like the terminal CLI on the homepage.

## Stack Choices

- **Astro** for static generation and content collections with Zod schema validation
- **React** (island) for the interactive terminal component — only hydrated on the client
- **Go + Gin** sidecar for dynamic API routes (contact form, analytics)
- **Docker** multi-stage build for production deployment
